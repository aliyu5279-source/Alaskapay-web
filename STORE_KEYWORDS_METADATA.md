## Metadata Optimization Tips

### 1. App Name Optimization
- **iOS**: Include primary keyword in subtitle
- **Android**: Include keyword in app name (within 50 chars)
- Example: "Alaska Pay - Digital Wallet"

### 2. Description Optimization
- Use keywords naturally in first 3 sentences
- Include keywords 3-5 times throughout
- Bold important features (Android only)
- Use bullet points for readability

### 3. Promotional Text (iOS)
- Update monthly with seasonal keywords
- Include current promotions
- Use action words (Download, Get, Try)

### 4. What's New Section
- Mention new features with keywords
- Keep it concise and scannable
- Update with every release

## ASO Best Practices

### Do's

### Don'ts

## Tracking & Analytics

### Key Metrics to Monitor
1. **Keyword Rankings**: Track position for top 20 keywords
2. **Conversion Rate**: Store listing visits → installs
3. **Search Impressions**: How often app appears in search
4. **Tap-Through Rate**: Search impressions → store visits
