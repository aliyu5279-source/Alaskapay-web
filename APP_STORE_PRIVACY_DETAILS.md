# App Store Privacy Details

## Apple App Store Privacy Nutrition Labels

### Data Used to Track You
**Financial Info**
- Payment Info: Used for processing transactions
- Purchase History: Used for transaction records

**Identifiers**
- User ID: Used to identify your account
- Device ID: Used for security and fraud prevention

### Data Linked to You
**Contact Info**
- Name: Required for account creation
- Email Address: Used for account verification and notifications
- Phone Number: Used for SMS verification and 2FA

**Financial Info**
- Payment Info: Required for money transfers and bill payments
- Credit/Debit Card Info: Stored securely for payment methods
- Purchase History: Transaction records
- Other Financial Info: Wallet balance, transaction history

**User Content**
- Photos: Optional for profile picture and KYC verification
- Other User Content: Support tickets, chat messages

**Identifiers**
- User ID: Account identification
- Device ID: Security and fraud detection

**Usage Data**
- Product Interaction: App usage analytics
- Advertising Data: Marketing campaign effectiveness

**Diagnostics**
- Crash Data: App stability improvements
- Performance Data: Optimization
- Other Diagnostic Data: Bug fixes

**Sensitive Info**
- Sensitive Info: KYC documents (passport, ID, utility bills)

### Data Not Linked to You
**Diagnostics**
- Crash Data: Anonymous crash reports
- Performance Data: Anonymous usage metrics

## Google Play Store Data Safety

### Data Collection & Security

#### Location
**Approximate location**
- Collected: Yes
- Shared: No
- Purpose: Fraud prevention, compliance
- Optional: Yes

#### Personal Info
**Name**
- Collected: Yes
- Shared: No
- Purpose: Account management
- Optional: No

**Email address**
- Collected: Yes
- Shared: No
- Purpose: Account verification, notifications
- Optional: No

**Phone number**
- Collected: Yes
- Shared: No
- Purpose: 2FA, account recovery
- Optional: No

**Address**
- Collected: Yes
- Shared: No
- Purpose: KYC verification
- Optional: Yes (for higher limits)

#### Financial Info
**Payment info**
- Collected: Yes
- Shared: Yes (with payment processors)
- Purpose: Process transactions
- Optional: No

**Purchase history**
- Collected: Yes
- Shared: No
- Purpose: Transaction records
- Optional: No

**Credit score**
- Collected: No

**Other financial info**
- Collected: Yes
- Shared: No
- Purpose: Wallet management
- Optional: No

#### Photos and Videos
**Photos**
- Collected: Yes
- Shared: No
- Purpose: Profile picture, KYC documents
- Optional: Yes

#### Files and Docs
**Files and docs**
- Collected: Yes
- Shared: No
- Purpose: KYC verification documents
- Optional: Yes

#### App Activity
**App interactions**
- Collected: Yes
- Shared: No
- Purpose: Analytics, improvements
- Optional: No

#### Device or Other IDs
**Device or other IDs**
- Collected: Yes
- Shared: No
- Purpose: Security, fraud prevention
- Optional: No

### Security Practices

**Data Encryption**
- Data is encrypted in transit: Yes (TLS 1.3)
- Data is encrypted at rest: Yes (AES-256)

**Data Deletion**
- You can request data deletion: Yes
- Account deletion available in app settings

**Independent Security Review**
- App has been independently validated: Yes
- Complies with PCI DSS standards

**Committed to Google Play Families Policy**
- No (app not designed for children)
