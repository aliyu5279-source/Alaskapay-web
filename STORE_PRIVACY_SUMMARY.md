## üîí Security Practices

### Data Encryption
**In Transit:**
- TLS 1.3 encryption
- Certificate pinning
- Secure API connections

**At Rest:**
- AES-256 encryption
- Encrypted database
- Secure key management

### Data Deletion
**User Rights:**
- Request data deletion anytime
- Account deletion in app settings
- Data removed within 30 days
- Backup deletion within 90 days

**Retention Policy:**
- Active accounts: Data retained
- Inactive 2+ years: Data archived
- Deleted accounts: 30-day grace period

- **SOC 2 Type II** - Security, availability, confidentiality
- **CBN Licensed** - Central Bank of Nigeria regulation

### Independent Security Review

- Quarterly security audits
