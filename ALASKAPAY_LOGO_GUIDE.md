## Usage Guidelines

### Size Recommendations
- **Navbar/Header**: 40-50px height
- **Footer**: 48-60px height
- **Favicon**: 32x32px or 64x64px
- **App Icons**: 192x192px, 512x512px
- **Social Media**: 1200x630px (use original)

### Spacing
- Always maintain 8-12px gap between logo and text
- Minimum clear space around logo: 16px
- Never stretch or distort the logo

### Color Variations
- Primary: Full color on white/light backgrounds
- Reversed: White version on dark backgrounds (if needed)
- Monochrome: Single color for special applications
